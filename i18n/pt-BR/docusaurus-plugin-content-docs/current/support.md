## Como Entrar em Contato com o Suporte Adyen

Se você não conseguiu resolver o problema que está enfrentando lendo a documentação, entre em contato com a equipe de Suporte Adyen. Siga as etapas abaixo para abrir um ticket na Adyen:

> **Importante**: Lembre-se de incluir os anexos listados abaixo do modelo, pois eles são essenciais para uma resolução rápida.

1.  **Prepare os detalhes do seu ticket**: Use o modelo a seguir para reunir todas as informações necessárias para uma resposta mais rápida:

    ```
    Assunto: Adyen V3 VTEX Connector + [Tópico relacionado ao seu problema]

    Descrição:
    Credenciais Adyen: [Sua Company Account, Sua Merchant Account] (Veja: [Obtendo Credenciais](./adyen-configuration/getting-credentials.md))
    Descrição do problema: [Texto descrevendo o comportamento]
    Detalhes da conta VTEX: [Nome da sua conta VTEX e nome da subconta (se houver)]
    Ambiente: [Teste/ Produção/ Ambos]
    Configuração de captura: [Tipo de captura configurada na VTEX: Captura Automática Imediatamente Após a Autorização do Pagamento/ Atraso de Liquidação Automática: 7 dias]
    Versão do Plugin: [Versão do plugin em uso, ex: v2.24.4]
    IDs VTEX: [ID do Pedido VTEX, ID da Transação VTEX] (Veja: [Monitoramento](./monitoring.md))
    TID / (Adyen) PSP Reference: [Informações de Pagamento/Oferta Adyen] (Veja: [Monitoramento](./monitoring.md))
    Chamadas de API Adyen: [Solicitações e Respostas de API Adyen]
    Webhooks Adyen: [Logs de Webhook de Autorização e Captura] (Veja: [Webhooks](./adyen-configuration/webhooks.md))

    Anexos:
    - Logs de Transação VTEX
    - Vídeo de Comportamento do Checkout
    - Prints de Erro
    - Logs de Rede (arquivo .HAR) - [Como gerar um arquivo HAR](https://help.vtex.com/pt/tutorial/como-gerar-arquivo-har--tutorials_4318)
    - Info de Debug do Navegador (sessão JAM) - [Como criar uma sessão JAM](https://jam.dev/)
    ```

2.  **Abra o ticket de suporte**: envie um e-mail para support@adyen.com.

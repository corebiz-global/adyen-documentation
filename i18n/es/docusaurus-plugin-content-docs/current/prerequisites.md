# Requisitos previos

Para seguir esta guía, necesitarás:

- Una cuenta VTEX con acceso de administrador.
- Una cuenta de Adyen.
- Sus credenciales de API de Adyen. Si aún no las tiene, vea cómo [generar su clave de API](./adyen-configuration/api-key.md).
